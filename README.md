# Lógica de Programação com JavaScript
Repositório criado para armazenar exercícios práticos de lógica de programação desenvolvidos durante meus estudos.

## Conteúdos abordados
- Variáveis e tipos de dados
- Condicionais
- Laços de repetição
- Funções
- Arrays e objetos
- Desafios práticos

## Objetivo
Consolidar a base lógica e aplicar conceitos fundamentais que serão utilizados em projetos front-end.
